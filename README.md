# StockPriceVolatilityForecasting
Stock price and volatility forecasting using ARIMA, GARCH(1,1), and GRU on NVDA, AAPL, and TSLA daily data (2010-2023). Includes feature engineering, winsorization, Min-Max/Standard scaling, and short vs long horizon evaluation. Built for IE 7275 Data Mining Engineering at Northeastern University, Spring 2026.
